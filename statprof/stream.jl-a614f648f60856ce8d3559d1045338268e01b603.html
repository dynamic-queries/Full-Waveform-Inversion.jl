192 priority_range()
193
194 Return the valid range of stream priorities as a `StepRange` (with step size 1). The lower
195 bound of the range denotes the least priority (typically 0), with the upper bound
196 representing the greatest possible priority (typically -1).
197 """
198 function priority_range()
199 least_ref = Ref{Cint}()
200 greatest_ref = Ref{Cint}()
201 cuCtxGetStreamPriorityRange(least_ref, greatest_ref)
202 step = least_ref[] < greatest_ref[] ? 1 : -1
203 return least_ref[]:Cint(step):greatest_ref[]
204 end

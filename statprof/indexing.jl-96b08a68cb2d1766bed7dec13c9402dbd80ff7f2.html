94 # simple array type that returns the index used to access an element, while
95 # retaining the dimensionality of the original array. this can be used to
96 # broadcast or reduce an array together with its indices, whereas normally
97 # combining e.g. a 2x2 array with its 4-element eachindex array would result
98 # in a 4x4 broadcast or reduction.
